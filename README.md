## 🧠 About This Project

* Built using **GitHub Pages (static hosting)**
* Each resume is an **HTML wrapper around a PDF**
* Designed for **quick sharing with recruiters**
* Fully accessible via direct links

---

## ⚡ Features

* 📱 Mobile-friendly viewing
* ⚡ Fast load (static assets)
* 🔗 Shareable role-specific links
* 📄 Embedded PDF preview

---

## 📌 Usage

You can directly share any resume link depending on the role you're applying for.

Example:

```text
Frontend Role → Use Frontend Resume Link
Backend Role → Use Backend Resume Link
```
